Используем 2 адаптера, но вместо Host-only:
Adapter 1 = Internal Network (рабочая сеть между VM) - статику 10.10.0.0/24
Adapter 2 = NAT (служебный “админский” доступ) - у каждой VM будет 10.0.2.x/24 (дефолтный NAT)
но внешний доступ с HOST делаем ТОЛЬКО на jumphost через проброс порта (port-forward)
остальные VM по SSH с HOST будут недоступны → только через jumphost → условие задания выполняется
