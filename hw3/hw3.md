Напишите скрипт по созданию 3 виртуальных машин и их первичной настройке (организуйте удаленный доступ по ssh и обновите пакеты)

Это может быть два отдельных скрипта (один создает виртуальные машины, а после установки ос вы настраиваете удаленный доступ и обновление репозиториев)

Варианты инструментов для решения:

1. VBoxManage + bash --> самое просто решение в лоб (показывал на занятии)
2. Vagrant -> как правило хороший инструмент для лабораторий
3. Terraform + VirtualBox provider --> для желающих быстрее погрузиться в terraform
   можете использовать все что для Вас удобно.


Цель задания привыкнуть к автоматизации всего того, что создаем.
